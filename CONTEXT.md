# School Recycling Rewards

This context defines the language for a school website where students exchange recyclable waste for reward points.

## Language

**นักเรียน**:
ผู้เรียนของโรงเรียนที่มีบัญชีสะสมแต้มและสามารถดูแต้มสะสมของตนเองได้.
_Avoid_: สมาชิก

**เลขประจำตัวนักเรียน**:
รหัสประจำตัวที่โรงเรียนกำหนดให้กับนักเรียนแต่ละคน และใช้เป็นข้อมูลสำหรับเข้าสู่ระบบของนักเรียน.
_Avoid_: ชื่อผู้ใช้, username

**รหัสผ่านเริ่มต้น**:
รหัสผ่านที่ระบบสร้างให้เมื่อตั้งค่าบัญชีนักเรียนครั้งแรก เพื่อให้ครูนำไปแจกให้นักเรียน.
_Avoid_: รหัสผ่านชั่วคราว

**ครู**:
ผู้ใช้ของโรงเรียนที่ดูแลข้อมูลนักเรียนและสร้างบัญชีนักเรียนในระบบ.
_Avoid_: อาจารย์, ผู้ดูแลระบบ

**ครูผู้ดูแลระบบ**:
บัญชีครูที่มีสิทธิ์สร้างบัญชีครูและเจ้าหน้าที่เพิ่มเติม โดยยังถือเป็นบทบาทครู ไม่ใช่สถานะบัญชีหลักแยกต่างหาก.
_Avoid_: แอดมินระบบ, super admin

**กติกาขยะ**:
ชุดของชนิดขยะและอัตราแต้มที่ครูผู้ดูแลระบบกำหนดให้เจ้าหน้าที่ใช้ตอนบันทึกการแลกขยะ.
_Avoid_: ตารางราคา, config แต้ม

**รายงานภาพรวม**:
ข้อมูลสรุปสำหรับครูเพื่อติดตามแต้มสะสม การแลกขยะ อันดับห้องเรียน และกราฟจากข้อมูลการแลกขยะ.
_Avoid_: แดชบอร์ดผู้ดูแลระบบ

**การนำเข้าข้อมูลนักเรียน**:
การเพิ่มข้อมูลนักเรียนจำนวนมากเข้าสู่ระบบจากไฟล์ตาราง เช่น CSV หรือ Excel.
_Avoid_: กรอกข้อมูลทั้งโรงเรียน

**ข้อมูลนักเรียน**:
ข้อมูลพื้นฐานที่ใช้สร้างบัญชีนักเรียน ได้แก่ เลขประจำตัวนักเรียน ชื่อ-นามสกุล ระดับชั้น และห้องเรียน.
_Avoid_: โปรไฟล์นักเรียนเต็มรูปแบบ

**ระดับชั้น**:
ระดับการศึกษาที่นักเรียนกำลังศึกษาอยู่ เช่น ม.1 หรือ ป.6.
_Avoid_: เกรด

**ห้องเรียน**:
กลุ่มชั้นเรียนที่นักเรียนสังกัด เช่น ม.1/1 หรือ ป.6/2 และใช้แบ่งอันดับแต้มของนักเรียน.
_Avoid_: กลุ่ม, คลาส

**เจ้าหน้าที่**:
ผู้ใช้ของโรงเรียนที่รับผิดชอบการบันทึกการแลกขยะของนักเรียน.
_Avoid_: แอดมิน, คนรับขยะ

**การแลกขยะ**:
เหตุการณ์ที่นักเรียนนำขยะรีไซเคิลมาส่งให้โรงเรียนและได้รับแต้มตอบแทน.
_Avoid_: ธุรกรรม, รายการซื้อขาย

**รายการแลกขยะ**:
บันทึกหนึ่งครั้งของการแลกขยะที่ผูกกับนักเรียนหนึ่งคน และอาจประกอบด้วยขยะหลายชนิดในรายการเดียว.
_Avoid_: รายการเดี่ยวต่อชนิดขยะ

**การตรวจทานรายการ**:
ขั้นตอนก่อนยืนยันรายการแลกขยะที่แสดงนักเรียน ขยะ จำนวน แต้ม และเศษคงค้างใหม่เพื่อให้เจ้าหน้าที่ตรวจความถูกต้อง.
_Avoid_: submit ทันที

**ประวัติการแลกขยะ**:
รายการย้อนหลังของการแลกขยะที่ส่งผลต่อแต้มของนักเรียน.
_Avoid_: log, transaction history

**แต้ม**:
หน่วยคะแนนสะสมที่นักเรียนได้รับจากการแลกขยะ.
_Avoid_: คะแนน, เงิน

**รายการปรับแก้แต้ม**:
รายการที่ใช้แก้ไขผลของการแลกขยะที่บันทึกผิด โดยต้องเก็บเหตุผลและผู้ทำรายการไว้เพื่อตรวจสอบย้อนหลัง.
_Avoid_: ลบรายการ, แก้แต้มเงียบ ๆ

**ชนิดขยะ**:
ประเภทของขยะรีไซเคิลที่โรงเรียนรับแลกและมีกติกาการให้แต้มของตัวเอง เช่น ขวดพลาสติกหรือฝาขวด.
_Avoid_: หมวดสินค้า

**อัตราแต้ม**:
กติกาที่กำหนดว่าขยะหนึ่งชนิดต้องใช้จำนวนกี่ชิ้นจึงได้หนึ่งแต้ม.
_Avoid_: ราคา, เรทเงิน

**เศษคงค้าง**:
จำนวนชิ้นของขยะชนิดหนึ่งที่นักเรียนส่งแล้วแต่ยังไม่ครบอัตราแต้ม และจะถูกนำไปรวมกับการแลกขยะครั้งถัดไปของนักเรียนคนเดิม.
_Avoid_: เศษแต้ม, แต้มทศนิยม

**อันดับห้องเรียน**:
ลำดับของนักเรียนในห้องเรียนเดียวกันตามแต้มสะสมรวม.
_Avoid_: leaderboard รวมทั้งโรงเรียน

**ฉลากพลาสติก**:
แผ่นพลาสติกที่หุ้มอยู่บนขวดพลาสติกและนับเป็นชนิดขยะแยกจากตัวขวด.
_Avoid_: แผ่นพลาสติก product
