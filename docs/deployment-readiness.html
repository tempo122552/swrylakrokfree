คู่มือเปิดใช้งานจริง

ระบบ `สว รย รักษ์โลกและสิ่งแวดล้อม` เวอร์ชันนี้เปลี่ยนไปใช้ Supabase Postgres แล้ว เพื่อให้ deploy ฟรีได้โดยไม่ต้องพึ่งไฟล์ฐานข้อมูล SQLite บนเครื่องหรือ persistent disk

ภาพรวม

  • เว็บรันบน Render Free
  • ฐานข้อมูลเก็บบน Supabase Postgres Free
  • ไม่ใช้ไฟล์ prisma/*.db แล้ว
  • Render ต้องตั้งค่า DATABASE_URL, DIRECT_URL, และ SESSION_SECRET

ขั้นตอนบน Supabase

  1. สมัครหรือเข้าสู่ระบบ Supabase
  2. สร้าง project ใหม่
  3. ไปที่ Project Settings > Database
  4. คัดลอก connection string แบบ pooled สำหรับ DATABASE_URL
  5. คัดลอก direct connection string สำหรับ DIRECT_URL

ขั้นตอนบน Render

  1. เข้า Render แล้วเลือก New > Blueprint
  2. เลือก repository tempo122552/swrylakrok
  3. Render จะอ่านไฟล์ render.yaml
  4. ตั้ง Environment Variables ให้ครบ
  5. รอ deploy จน Live
  6. เปิด /api/health เพื่อตรวจระบบ

Environment Variables

DATABASE_URL="postgresql://postgres.[PROJECT_REF]:[PASSWORD]@...pooler.supabase.com:6543/postgres?pgbouncer=true&connection_limit=1"
DIRECT_URL="postgresql://postgres:[PASSWORD]@db.[PROJECT_REF].supabase.co:5432/postgres"
SESSION_SECRET="ค่าสุ่มยาวอย่างน้อย 32 ตัวอักษร"

หลังเปิดใช้งานครั้งแรก

  • login ครูด้วย teacher.admin
  • รหัสผ่านเริ่มต้นคือ Password123!
  • เปลี่ยนรหัสผ่านทันที
  • สร้างบัญชีเจ้าหน้าที่จริง
  • import รายชื่อนักเรียนทั้งโรงเรียน
